# 家政管理系统 python454

## 项目概述

家政管理系统是基于Python Django框架和MySQL数据库构建的一套业务处理系统。该系统旨在为家政服务提供商提供一个便捷、高效的管理平台。

## 技术栈

- **后端**: Python Django框架
- **数据库**: MySQL
- **推荐开发环境**: PyCharm

## 功能模块

- 用户管理
 - 注册与登录
 - 权限控制
- 服务管理
 - 服务发布
 - 服务预约
- 订单管理
 - 订单创建
 - 订单跟踪
- 费用管理
 - 费用计算
 - 费用记录

## 系统角色

- 系统管理员
- 家政服务提供者
- 客户

## 运行环境

- Python 3.6 或以上版本
- Django 2.2 或以上版本
- MySQL 5.7 或以上版本

## 部署步骤

1. 安装Python及Django环境
2. 配置MySQL数据库
3. 导入数据库结构
4. 运行Django项目
5. 打开浏览器访问系统
